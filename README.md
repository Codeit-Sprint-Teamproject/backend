# 모읽지(Moilji)_BACKEND
![img_3.png](img_3.png)
코드잇 스프린트 5기 협업 프로젝트를 통해 진행한 독서 커뮤니케이션 웹 서비스 서버 코드 입니다.

# 프로젝트 소개
모읽지(Moilji) 는 독서 모임과 커뮤니티 기능을 제공하는 플랫폼으로, 사용자들이 함께 책을 읽고 토론하며 성장할 수 있는 환경을 제공합니다.

+ Database: MySql(AWS RDS)

+ NoSql: Redis
+ CI/CD: GitHub Actions
